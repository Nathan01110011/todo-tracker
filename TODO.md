# TODO Tracker - Future Roadmap

This file tracks ideas and features for future versions of the TODO Tracker.

## 📸 Photos & Media
- [ ] **Photo Uploads**: Add the ability to attach photos to visited locations.
  - *Tech Suggestion*: Use **Cloudinary** for easy uploads and automatic image optimization.
  - *UI*: Show thumbnails in the "Visited" list and in map popups.
- [ ] **Gallery View**: A dedicated tab to browse all photos taken during adventures.

## 📍 Mapping & Location
- [ ] **Distance Badges**: Show "5 min drive" or "2.4 miles away" next to each location relative to your "Home" marker.
- [ ] **Automated Routes**: Generate an optimized route/itinerary for a day of activities.
- [ ] **Custom Marker Colors**: Color-code markers by category (e.g., Green for Food, Purple for Sport, Blue for Drinks).
- [ ] **Street View Integration**: Add a link or mini-window to see the Google Street View of a location.

## 🔍 UX & Search Improvements
- [ ] **Search on Map**: Add a search bar directly on the map to find nearby points of interest.
- [ ] **Sort by Proximity**: Sort the "To Do" list by what's currently closest to you.
- [ ] **Delete functionality**: Add a "Remove" button for locations (currently requires manual spreadsheet deletion).

## 🛠️ Tech & Infrastructure
- [ ] **Export/Backup**: A button to export the current data as a CSV (beyond the Google Sheet).
- [ ] **Offline Mode**: Cache the data locally so it can be viewed without an internet connection.
- [ ] **Multi-user Support**: Allow switching between different tracker "profiles" or shared sheets.
